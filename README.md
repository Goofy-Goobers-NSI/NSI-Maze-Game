# NSI-Maze-Game
Un jeu où le joueur doit faire la course contre un algorithme pour trouver la sortie d'un labyrinthe généré aléatoirement, développé avec le module pygame-ce.

## Installation et Lancement

1. **Prérequis** : Avoir **Python 3.12** (ou supérieur) installé.
2. **Dépendances** : Le projet utilise la bibliothèque `pygame-ce`. 
Pour l'installer :
- pip install pygame-ce
3. **Exécution** : Ouvrir le dossier src et lancer le fichier main.py :
- python main.py

### Menu Principal
* **Navigation** : Utilisez la souris pour cliquer les boutons.
* **Sélection** Cliquez pour choisir un mode de jeu, voir le classement ou accéder aux paramètres.

### Contrôles en jeu
* **Déplacement** : Touches directionnelles (les fleches par defaut).
* **Paramètres** Les touches sont modifiables dans le menu (ZQSD, WASD ou ARROWS). Vous pouvez egalement changer la difficulter de l'IA dans le même menu, pour le mode de jeu Versus.
